

# InfoBrains Website

Official website for **InfoBrains Club April Event**, providing registration, verification, and secure access for users.

## 📁 Project Structure

```
C:.
│   index.html
│   logoInfoB.jpg
│   posterTemplateInfobrainsClub.pdf
│   proxy.php
│   scriptlp.js
│   styleLP.css
│
└───registre
        index.php
        verify.php
```

## 🚀 Features

* Landing page for InfoBrains Club
* User registration system
* Verification system for submitted data
* Proxy handling via PHP
* Clean UI with custom styling

## 🔐 Security

This project includes multiple layers of protection:

* Cloudflare Anti-Bot protection
* AES encryption for sensitive data
* Payload hashing to prevent tampering
* JavaScript obfuscation for client-side protection

## 📄 Pages Overview

### `index.html`

Main landing page of the website.

### `scriptlp.js`

Handles client-side logic, encryption, and interactions.

### `styleLP.css`

Custom styles for the landing page.

### `proxy.php`

Acts as a secure proxy between client and backend.

### `/registre/index.php`

Registration page where users submit their information.

### `/registre/verify.php`

Verification system to validate user data.
